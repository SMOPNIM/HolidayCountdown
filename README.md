# 距离放假还有…

ClassIsland 插件，在主界面显示距离下一个法定节假日还有多久，支持调休补班自动创建临时层。

## 该插件已经弃用
 由于本插件功能已经有其他插件实现，所以弃用。

## 功能

- **倒计时显示** — 在主界面组件显示距离下一个节假日的倒计时，可自定义格式字符串（`%D` 总天数、`%H` 总小时、`%M` 总分、`%S` 总秒、`%N` 节日名称等）
- **多数据源** — 支持本地内置数据、GitHub 远程 JSON、timor.tech API 三种来源，可切换为「Both（API 优先）」模式自动获取
- **内置数据** — 内置 2025–2028 年法定节假日及调休补班数据，含农历节日（春节、端午、中秋）
- **远程自动更新** — 从远程源获取最新节假日安排，缓存到本地，后续启动优先读取缓存
- **调休补班自动临时层** — 在调休上班日根据目标节假日的星期几自动匹配课表并创建临时层，无需手动配置课表 ID
- **调休补班纳入倒计时** — 可选是否将调休补班日显示在倒计时中
- **节假日管理** — 在插件设置页中查看、添加、编辑、删除节假日条目
- **显示自定义** — 每个组件实例独立设置：紧凑模式、强调色、字体大小、连接词着色
